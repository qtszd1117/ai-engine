🔌 插件管理

加载JavaScript插件来扩展功能。插件可以在消息发送前/后、提示词构建时等关键节点注入自定义逻辑。

📋 可用钩子说明

onBeforeSend(text)
用户发送消息前触发。参数是用户输入文本，返回修改后的文本。
return text + " (已修改)";
onAfterReceive(text)
收到AI回复后触发。参数是AI回复文本，返回修改后的文本。
return text.replace(/坏词/g, "***");
onBuildPrompt(messages)
构建提示词后、发送前触发。参数是消息数组，可以修改/添加/删除消息。
messages.push({role:"system", content:"额外指令"});
return messages;
onMessageDisplay(element, msg)
消息显示到界面后触发。element是DOM元素，msg是消息数据。
可以用来添加按钮、修改样式等。
onChatLoad(chat)
切换/加载对话时触发。参数是当前对话对象。
onInit()
插件首次加载时触发。可以用来初始化插件状态、注册UI等。
