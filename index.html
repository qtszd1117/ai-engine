📋 可用钩子说明

onBeforeSend(text)
用户发送消息前触发。参数是用户输入文本，返回修改后的文本。
return text + " (已修改)";
onAfterReceive(text)
收到AI回复后触发。参数是AI回复文本，返回修改后的文本。
return text.replace(/坏词/g, "***");
onBuildPrompt(messages)
构建提示词后、发送前触发。参数是消息数组，可以修改/添加/删除消息。
messages.push({role:"system", content:"额外指令"});
return messages;
onMessageDisplay(element, msg)
消息显示到界面后触发。element是DOM元素，msg是消息数据。
可以用来添加按钮、修改样式等。
onChatLoad(chat)
切换/加载对话时触发。参数是当前对话对象。
onInit()
插件首次加载时触发。可以用来初始化插件状态、注册UI等。

📚 快速开始

第1步：配置API
点击侧边栏「🔌 API设置」→ 选择API类型 → 填入地址和密钥 → 点击测试
第2步：创建或导入角色卡
点击「🎭 角色卡」→ 新建单人/多人角色 → 填写角色设定
也可以用「🔄 酒馆导入」导入现成的角色卡
第3步：配置对话
在对话页面展开「📜 提示词/绑定」→ 绑定角色卡 → 选择预设模板
也可以绑定面具（用户人设）和世界书
第4步：开始对话
在输入框输入消息 → 按发送 → 等待AI回复
可以用 ▶ 按钮生成新回复，◀ ▶ 切换不同版本

🎵 音乐与外观

🎵 音乐歌单：在「🎨 外观」中添加音乐 → 创建歌单 → 绑定到对话
🖼️ 背景图：每个对话可以设置不同的背景图
📝 字体：可以导入在线字体，每个对话绑定不同字体
🎨 主题：一键切换主题包，或自定义CSS

💡 进阶技巧

📖 世界书：设置关键词触发的背景信息，AI会在合适时机自动引用
📝 自动总结：开启后会自动总结早期对话，防止角色遗忘（OOC）
🔤 正则：自动过滤或替换AI回复中的特定内容
🔀 分支：点击消息的 🔀 按钮可以从任意节点创建分支对话
💾 存档：定期导出备份，防止数据丢失

❓ 常见问题

API连接失败怎么办？
1. 检查地址末尾是否有多余的 /
2. 检查密钥是否正确
3. 中转站选「OpenAI（代理/中转）」类型
4. 确认网络连接正常
AI回复被截断怎么办？
在API参数中调大「最大回复」数值，建议设为 4096 或更高。
怎么导入酒馆的角色卡？
点击侧边栏「🔄 酒馆导入」，选择 JSON 文件即可。支持角色卡、预设、世界书、正则等。
数据存在哪里？安全吗？
所有数据存在你的浏览器本地（IndexedDB），不会上传到任何服务器。建议定期在「💾 存档」中导出备份。
换设备后数据怎么迁移？
在旧设备：💾存档 → 📤导出全部
在新设备：💾存档 → 📥导入 → 选择导出的文件
